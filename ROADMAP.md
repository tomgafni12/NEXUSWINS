# NexusWins Roadmap

## Phase 1 — Foundation (Current)
- [ ] Set up Arch Linux base in VirtualBox
- [ ] Get familiar with archiso (custom ISO builder)
- [ ] Build a bootable NexusWins ISO for the first time

## Phase 2 — Compatibility Layers
- [ ] Integrate Wine (Windows app support)
- [ ] Integrate Waydroid (Android app support)
- [ ] Integrate Darling (macOS app support)
- [ ] Test running apps from each platform

## Phase 3 — User Experience
- [ ] Build a unified app launcher
- [ ] Custom installer so anyone can install NexusWins
- [ ] Custom branding and desktop theme

## Phase 4 — Community
- [ ] First public release (alpha)
- [ ] Documentation site
- [ ] Accept contributions from the community

## Long Term Goals
- Custom UI/desktop environment
- Hardware support (prebuilts)
- Embedded systems
- SoC development

## The Dream
An OS where it doesn't matter what platform an app was 
built for. It just works.
