# Contributing to NexusWins

> "There are nexuses, causal relationships that cannot 
> be separated." — Madara Uchiha

Welcome! NexusWins is an open source project and 
contributions are always welcome.

## Getting Started
1. Fork the repository
2. Clone your fork
3. Create a branch for your feature
4. Make your changes
5. Commit & push
6. Open a pull request

## What We Need Help With
- Arch Linux packaging
- Wine integration
- Waydroid integration
- Documentation
- Testing

## Rules
- Be respectful
- Write clear commit messages
- Document your code
- Test before submitting

## Contact
Open an issue on GitHub for questions or ideas.
