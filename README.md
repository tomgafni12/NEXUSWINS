# NexusWins

> *"There are nexuses, causal relationships that cannot be separated. 
> A world of only victors."* — Madara Uchiha

NexusWins is an open source Linux-based OS built on one idea: 
every app from every platform should just work.

## What is NexusWins?

An Arch-based Linux distribution with built-in compatibility 
layers so you never have to worry about what OS an app was 
made for.

- Windows app? Works.
- Android app? Works.
- Linux app? Obviously works.

## Goals

- [ ] Base Arch Linux install
- [ ] Wine integration (Windows apps)
- [ ] Waydroid integration (Android apps)
- [ ] Darling integration (macOS apps)
- [ ] Unified app launcher
- [ ] Custom installer
- [ ] Custom UI
